## 我的小站

基于Hexo引擎的博客，使用[hexo-theme-cola主题](https://github.com/Aizener/hexo-theme-cola)

## 如何开启服务

首先使用pnpm安装依赖

```bash
pnpm install
```

然后执行以下命令启动服务

```bash
hexo clean & hexo g & hexo s
```
